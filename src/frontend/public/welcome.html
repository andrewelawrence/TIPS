What it Does

TIPS helps you understand the nuance, tone, and implied meaning in online text, images, or links using AI. You can interpret content directly or provide specific context from the page for a more accurate analysis.

How to Use

1. Interpret Content

Right-click on selected text, an image, or a link, then choose "Interpret" from the context menu.

Alternatively, for selected text, you can still click the small ðŸ’¡ icon that appears.

The AI will analyze the content (and any added context, see below). Click the TIPS extension icon (ðŸ’¡) in your browser toolbar to see the interpretation, tone, and confidence score.
(Pin the icon to your toolbar for easier access!)

2. Adding Context

Sometimes, the AI needs more background information. Right-click on relevant text snippets, images, or links on the page and choose "Add to Context".

These items will be included with your next "Interpret" request on that same tab, helping the AI understand the situation better.

You can clear the context at any time by right-clicking on the TIPS icon in the toolbar and selecting "Clear Context".

ðŸš€ Important Setup: API Key

TIPS requires an API key from Anthropic to function. You need to add this key *manually*.

  1. Obtain an API key from the Anthropic Console.
  2. If you installed this extension from source code (developer mode):
    • Navigate to the `src/frontend` directory in your project folder.
    • Create a file named .env.local (if it doesn't exist).
    • Add your key like this: VITE_ANTHROPIC_API_KEY=YOUR_KEY_HERE
    • Rebuild the extension (npm run build in `src/frontend`) and reload it in Chrome (chrome://extensions/).

Without a valid API key configured, the interpretation feature will not work.
