Thank you for choosing TIPS! TIPS helps you understand the nuance, tone, and implied meaning in online text, images, or links using AI-powered analysis.

What it Does

TIPS uses AI to help interpret online content, making communication clearer. You can either interpret content directly or provide specific context from the page for a more accurate analysis.

How to Use

1. Interpret Text

Right-click on selected text, then choose "Interpret" from the context menu. Alternatively, click the small ðŸ’¡ icon that appears next to selected text.

Interpreting text content with TIPS

2. Interpret Images

Right-click on an image and select "Interpret" from the context menu. TIPS will analyze the visual content for you.

Interpreting image content with TIPS

3. Adding Context

Sometimes, the AI needs more background information. Right-click on relevant text snippets, images, or links on the page and choose "Add to Context". These items will be included with your next "Interpret" request on that same tab.

Adding context to improve interpretation accuracy

You can clear the context at any time by right-clicking on the TIPS icon in the toolbar and selecting "Clear Context". Context is also automatically cleared when tabs are closed.

Enjoy using TIPS! We hope it helps make online communication clearer.
